Ordered n-tuples are often indicated by a variable with an arrow on top. For example, we can let

$$
c = \pm\sqrt{a^2 + b^2}
$$

Lift($L$) can be determined by Lift Coefficient ($C_L$) like the following equation.

$$
L = \frac{1}{2} \rho v^2 S C_L
$$
